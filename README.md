# pi-cc-header

[中文](#中文)

Animated Pi logo header for [pi coding agent](https://pi.dev).

## Features

- 14-frame pixel-animated Pi logo from pi.dev/install.sh
- Precomputed frames for zero-cost rendering
- Toggleable IBM-style horizontal stripes (`/hl`)
- 7-color palette: red, orange, yellow, green, blue, purple, white (`/hc`)
- Version number color toggle (`/hv`)
- Displays Pi version, model, thinking level, extension/skill counts, and cwd

## Credits

Logo animation adapted from [pi.dev/install.sh](https://pi.dev/install.sh).

## 功能

- 14 帧像素 Pi logo 动画（取自 pi.dev/install.sh）
